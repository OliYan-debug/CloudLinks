
# Cloud Links


This project is for the people who always have a link to save but sometimes forget where they are and for what is that link


## Lessons Learned

This project was a great challenge because the [M.V.C model](https://en.wikipedia.org/wiki/Model%E2%80%93view%E2%80%93controller) of programming was new to me but all the implementation was focus in simplicity with the goal of have a applications working with these practices

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PASSWORD`


## Tech Stack

**Client:** Ejs

**Server:** Node, Express, MongoDB


## Features

- Save your links anywhere
- Easy Access to your link siteurl/title
- Mobile Friendly
